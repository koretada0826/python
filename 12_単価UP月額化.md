## 1. 受注後の動き方(30分)

### Step 1:24時間以内にヒアリング

クロードに頼む:
```
ランサーズ案件のヒアリングメッセージ書いて。

案件:
<コピペ>

確認したいこと:
- データ件数の想定
- 出力形式の希望
- 例外時の挙動

トーン: 丁寧、相手の負担にならない
```

### Step 2:中間報告(全体50%時点)

```
〇〇様

進捗のご報告です。

▼ 完了
- ファイル読み込み機能
- データ抽出ロジック

▼ 残り
- エラーハンドリング
- ドキュメント整備

予定通り、明日納品させていただきます。
```

### Step 3:納品

ZIPで:
```
project.zip
├── main.py
├── README.md
├── requirements.txt
├── sample_input/
└── sample_output/
```

クロードに「納品メッセージ書いて」で完璧な文章作れる。

## 2. 評価★5の取り方(20分)

5要素:

1. **24時間以内に必ず返信**
2. **わかりやすいREADME**
3. **想定外を先回り**(エラー処理しっかり)
4. **修正に快く応じる**(2回まで無料が普通)
5. **最後の一言**:「またのご機会あれば是非お願いします」

→ これで★5率90%。

### トラブル対処

| 問題 | 対応 |
|---|---|
| 納期間に合わない | **早めに**(3日前)相談 |
| 検収されない | 14日経過で**自動検収** |
| クレーマー | ランサーズ運営に通報、証拠保存 |

## 3. 単価を上げる4段階(30分)

### 段階表

| ★5の数 | 単価倍率 |
|---|---|
| 0-3件 | 半額(実績作り) |
| 3-10件 | 通常 |
| 10-30件 | 1.5倍 |
| 30件超 | 2倍以上 |

### 単価表を持つ

プロフィールに貼る:

```markdown
## 価格表

### 単発
- 簡単な自動化(50行程度): 8,000円〜
- データ集計・グラフ化: 20,000円〜
- スクレイピング(1サイト): 30,000円〜
- AI連携アプリ: 80,000円〜
- RAG検索Bot: 200,000円〜

### 月額契約
- ライト: 30,000円/月(月10時間)
- スタンダード: 50,000円/月(月20時間)
- プレミアム: 100,000円/月(月50時間)
```

→ **値引き要求の盾**になる。

---

## 4. ⭐ 月額契約に持ち込む(40分)

### 案件1件 vs 月額契約

| 単発(30,000円×5件) | 月額(30,000円×5社) |
|---|---|
| 毎月新規営業必要 | 自動で入る |
| 不安定 | **安定** |

→ 月額契約**3〜5社作れば月10〜30万円安定**。

### 提案タイミング

納品時に投げる:

```
このツール、運用にあたり以下のサポートが必要かと思います。

▼ 月額サポート
- ツール稼働の月次監視
- 軽微な修正・改善
- メールでの相談無制限

月額: 30,000円〜

ご導入企業様の声:
「自動化のおかげで人件費が月10万削減」

ご検討ください。
```

→ 1社2社契約取れれば**月収が安定**する。
