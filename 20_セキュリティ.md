世界で**最重要**とされる10大脆弱性。

| # | 脆弱性 | 例 |
|---|---|---|
| 1 | **アクセス制御の不備** | 他人のデータが見れる |

## 2. パスワード保護(15分)

### NG(絶対やるな)

```python
# ❌ 平文保存
user.password = "tarou123"

# ❌ MD5 / SHA1
import hashlib
user.password = hashlib.md5(b"tarou123").hexdigest()
```

### OK

```python
from passlib.context import CryptContext
pwd_context = CryptContext(schemes=["bcrypt"])

# ハッシュ化(保存)
user.password_hash = pwd_context.hash("tarou123")

# 検証(ログイン)
if pwd_context.verify("tarou123", user.password_hash):
    print("OK")
```

→ **bcrypt or argon2**を使う。MD5/SHA1 は禁止。

## 3. SQLインジェクション(20分)

### 危険コード

```python
# ❌ 文字列埋め込み(SQLi 食らう)
cursor.execute(f"SELECT * FROM users WHERE name = '{name}'")
```

→ ユーザーが `' OR 1=1; --` と入れたら**全データ奪われる**。

### 安全コード

```python
# ✅ プレースホルダ
cursor.execute("SELECT * FROM users WHERE name = ?", (name,))

# SQLAlchemy ORM
db.query(User).filter(User.name == name).all()

# SQLAlchemy text + params
db.execute(text("SELECT * FROM users WHERE name = :name"), {"name": name})
```

→ **絶対に文字列連結しない**。プレースホルダ必須。

## 4. XSS(クロスサイトスクリプティング)(15分)

「**他人のブラウザで悪意あるJSを実行**」される攻撃。

### 危険(Webアプリ)

```python
# ❌ ユーザー入力をそのままHTML出力
return f"<h1>こんにちは {name}</h1>"
# → name に <script>alert(1)</script> 入ると実行される
```

### 安全

```python
# ✅ Jinja2 テンプレート(自動エスケープ)
return render_template("hello.html", name=name)

# Streamlit/FastAPI の通常使用は安全(JSON返すだけ)
```

### markdown を扱う時の罠

```python
# ❌ Markdown → HTML 変換時の生HTML許可
markdown.markdown(text, extensions=["html"])

# ✅ 生HTML 禁止 + サニタイズ
import bleach
clean = bleach.clean(html, tags=["p", "b", "i"], strip=True)
```

## 5. CSRF(クロスサイトリクエストフォージェリ)(10分)

「他人のサイトから自分のサイトに**勝手なリクエスト**送られる」。

### 防御

- POST/PUT/DELETE は **CSRFトークン**で保護
- FastAPI なら `fastapi-csrf-protect` 使う
- API なら **JWT + Authorization ヘッダー**で十分

## 6. 機密情報の管理(20分)

### NG

```python
# ❌ ハードコード
API_KEY = "sk-xxxxxxxxxxxx"

# ❌ Git にコミット
```

### OK

```python
# ✅ .env ファイル
import os
from dotenv import load_dotenv
load_dotenv()
API_KEY = os.getenv("API_KEY")
```

`.gitignore` に必ず:
```
.env
.env.local
*.pem
secrets/
credentials.json
```

### 本番(クラウド)

| サービス | 機密管理 |
|---|---|
| AWS | Secrets Manager / Parameter Store |
| Render | 環境変数(UI入力) |
| Heroku | Config Vars |

→ **環境変数経由**が世界標準。

### Git に上げてしまった時

```bash
# 履歴から完全削除(BFG Repo-Cleaner)
brew install bfg
bfg --replace-text passwords.txt

# 古いキーは即無効化(Anthropic/OpenAI管理画面で)
```

→ **見つけたら即無効化**。流出キーは数分で悪用される。

## 7. 認可(認証じゃない)(15分)

| 用語 | 意味 |
|---|---|
| **認証(Authentication)** | あなたは誰? |
| **認可(Authorization)** | 何できる? |

### 基本

```python
@app.get("/admin")
def admin_panel(user = Depends(get_current_user)):
    if not user.is_admin:
        raise HTTPException(403, "管理者のみ")
    return {...}

@app.delete("/users/{id}")
def delete_user(id: int, user = Depends(get_current_user)):
    if user.id != id and not user.is_admin:
        raise HTTPException(403, "他人のアカウント削除禁止")
    ...
```

→ **「自分のデータ以外見れない」**を必ず実装。

## 8. レート制限(15分)

「**1秒に100回叩かれる**」攻撃から守る。

```bash
pip install slowapi
```

```python
from slowapi import Limiter
from slowapi.util import get_remote_address

limiter = Limiter(key_func=get_remote_address)
app.state.limiter = limiter

@app.get("/api/data")
@limiter.limit("10/minute")
def get_data(request: Request):
    return {...}
```

→ 1分10回まで。超過は 429 Too Many Requests。

## 9. HTTPS(15分)

### 必須

- 本番は**必ず HTTPS**
- HTTP は盗聴される

### 取得方法

| 方法 | 価格 |
|---|---|
| Let's Encrypt | 無料 |
| Cloudflare | 無料(DNS経由) |
| AWS ACM | 無料 |

### 設定

```python
# FastAPI のリダイレクト
from fastapi.middleware.httpsredirect import HTTPSRedirectMiddleware
app.add_middleware(HTTPSRedirectMiddleware)
```

→ Render/Vercel/Cloudflare はデフォルトでHTTPS有効。

## 11. ロギングの注意(10分)

### NG

```python
# ❌ ログにパスワード残す
logger.info(f"Login attempt: {email}, {password}")

# ❌ 機密データをログに
logger.info(f"User data: {user.dict()}")  # ← password_hash含むかも
```

### OK

```python
# ✅ 機密は除外
logger.info(f"Login attempt: {email}")

# ✅ ログ用 dict
def safe_user(user):
    return {"id": user.id, "email": user.email}  # passwordなし

logger.info(f"User: {safe_user(user)}")
```

## 13. PII(個人情報)取扱(15分)

日本: **個人情報保護法**、EU: **GDPR**

### 守ること

- 必要最小限のデータだけ取得
- 暗号化保存(可能なら)
- 削除請求に応える(忘れられる権利)
- 第三者提供時は明示・同意

### Python での暗号化

```bash
pip install cryptography
```

```python
from cryptography.fernet import Fernet

key = Fernet.generate_key()
f = Fernet(key)

encrypted = f.encrypt(b"機密情報")
decrypted = f.decrypt(encrypted)
```
