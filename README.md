MkDocsでサイト化済み。Render などでデプロイすれば検索付きの綺麗な講座サイトになる。

### ローカルで確認

```bash

mkdocs serve

```

### Render にデプロイ

1. New Static Site
2. Repo: `koretada0826/python`
3. **Build Command**: `pip install -r requirements.txt && mkdocs build`
4. **Publish Directory**: `site`
