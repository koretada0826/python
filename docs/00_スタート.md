# 🐍 月100万Pythonエンジニア最短コース

## このコースの目的

**月収100万円のフリーランス**を**最短12日**で立ち上げる学習プログラム。

---

## 月100万円の構成(現実解)

時給5,000円 × 200時間 = 100万円(SES会社員的な働き方なら可能)
…ではなく、**フリーランスとして100万円稼ぐ正しい設計**は:

```
月額契約 5〜8社 × 50,000〜100,000円 = 30〜80万円(安定)
+ 単発高単価案件(AI開発) 100,000〜500,000円 × 2〜4件 = 30〜80万円
+ 紹介・継続(リピート率80%) = 自動的に上乗せ

= 月100万円〜
```

**労働時間 月100〜150時間で達成**(週稼働25-35h)。1日6h未満でOK。

---

## 鉄則(ここがズレると100万到達しない)

1. **時給を売らない** ← 上限が出る
2. **価値・成果物を売る** ← ROIで説得
3. **月額契約を必ず作る** ← 安定の柱
4. **領域を絞る** ← EC・士業・医療など、専門家になる
5. **AI連携できる** ← 単価3-5倍
6. **SES または直接契約** ← 中抜き手数料20%回避
