# Day 1:環境構築 + Hello, World

## 💰 この章後の月収:0円(準備)
## ⏱ 所要:2時間

---

## ゴール

- ターミナルでPythonが動く
- VS Code でコード書ける
- GitHub アカウント持つ
- "Hello"画面に出せる

---

## ステップ1:Python確認(10分)

ターミナル開く: `Cmd + Space` → 「ターミナル」

```
python3 --version
```

→ `Python 3.x.x` が出ればOK。出なかったら https://python.org からインストール。

---

## ステップ2:VS Code(20分)

1. https://code.visualstudio.com/ から **Download for macOS**
2. zip解凍 → `Visual Studio Code.app` を **アプリケーション** にドラッグ
3. 起動

### 必須拡張(左の田んぼマーク)
- **Python**(Microsoft製)
- **Japanese Language Pack**(日本語化)

→ 入れたら再起動。

---

## ステップ3:作業フォルダ(10分)

ターミナルで:
```
mkdir ~/python_lesson
cd ~/python_lesson
```

VS Code で `~/python_lesson` を開く(ファイル → フォルダーを開く)。

---

## ステップ4:Hello, World(15分)

VS Codeで `hello.py` 新規作成。中身:

```python
print("Hello, World!")
print(2 + 3)
print("私は" + " " + "Pythonエンジニア")
```

`Cmd + S` 保存 → 右上 ▶ ボタンで実行。

```
Hello, World!
5
私は Pythonエンジニア
```

→ **🎉 初Python動いた**。

---

## ステップ5:GitHub アカウント(20分)

1. https://github.com で Sign up
2. ターミナルで:
```
git config --global user.name "あなたの名前"
git config --global user.email "登録したメアド"
```

---

## ステップ6:Claude(クロード)準備(15分)

1. https://claude.ai でアカウント
2. **Pro契約($20/月)推奨**(無料版だと制限多い)

→ これがあなたの**永久に使える相棒**になる。

## つまづき対処

| 問題 | 解決 |
|---|---|
| `python3: command not found` | python.org からインストール、ターミナル開き直し |
| VS Codeで実行ボタンない | Python拡張入れ直し |
| 認証失敗 | クロードに「<エラー文> 直して」 |

→ **困ったら全部クロードに丸投げ**。
