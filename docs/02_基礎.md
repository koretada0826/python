## 1. 変数(5分)

「箱に名前つける」だけ。

```python
名前 = "太郎"        # 文字(str)
年齢 = 25            # 整数(int)
身長 = 170.5         # 小数(float)
学生 = True          # 真偽(bool)

print(f"私は{年齢}歳の{名前}です")
# → 私は25歳の太郎です
```

`f"..."` の中に `{変数}` を入れると展開される(超便利)。

## 2. if(条件分岐)(15分)

```python
点数 = 75

if 点数 >= 90:
    print("S")
elif 点数 >= 70:
    print("B")
else:
    print("不合格")
```

**インデント(スペース4つ字下げ)** が必須。

### 比較演算子
| 記号 | 意味 |
|---|---|
| `==` | 等しい |
| `!=` | 等しくない |
| `>=` | 以上 |
| `<=` | 以下 |
| `and` | かつ |
| `or` | または |
| `not` | ではない |

## 3. for(繰り返し)(15分)

```python
# 1〜10を表示
for i in range(1, 11):
    print(i)

# リストを回す
果物 = ["りんご", "みかん", "バナナ"]
for f in 果物:
    print(f)

# 合計を出す
合計 = sum([10, 20, 30, 40])
print(合計)  # 100
```

---

## 4. リスト(15分)

```python
果物 = ["りんご", "みかん", "バナナ"]

果物[0]               # "りんご" (1番目=0)
果物[-1]              # "バナナ" (最後)
果物.append("ぶどう") # 末尾追加
果物.remove("みかん") # 削除
len(果物)             # 個数
```

---

## 5. 辞書(15分)

```python
人 = {
    "名前": "太郎",
    "年齢": 25
}

人["名前"]             # "太郎"
人["メール"] = "x@y.z" # 追加
人.get("住所", "不明")  # キーないと"不明"返す

# 全部回す
for key, value in 人.items():
    print(f"{key}: {value}")
```

## 6. 関数(20分)

```python
def 税込(税抜, 率=0.10):
    return int(税抜 * (1 + 率))

合計 = 税込(1000)        # 1100
合計2 = 税込(1000, 0.08) # 1080
```

`def 関数名(引数):` → 中身は字下げ → `return` で値返す。
