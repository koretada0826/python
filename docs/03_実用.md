## 1. エラーは敵じゃない(20分)

エラー = **「ここをこう直して」のメッセージ**。

### よく出るエラー一覧

| エラー名 | 意味 | 解決 |
|---|---|---|
| `NameError` | 変数名違う | スペル確認 |
| `TypeError` | 型違う | int/str変換 |
| `IndexError` | リスト範囲外 | 範囲内へ |
| `KeyError` | 辞書のキーない | `.get()` 使う |
| `IndentationError` | インデント違う | スペース4つ |
| `FileNotFoundError` | ファイルない | パス確認 |
| `ZeroDivisionError` | 0で割った | 条件チェック |

### try/except でクラッシュ防ぐ

```python
try:
    数 = int(input("数字: "))
    print(10 / 数)
except ValueError:
    print("数字じゃない")
except ZeroDivisionError:
    print("0で割れない")
except Exception as e:
    print(f"エラー: {e}")
```

→ **try で囲めばプログラム止まらない**。

### 解決の魔法

```
このエラー出た。直して:

コード:
<コピペ>

エラー:
<コピペ>
```

→ クロードに丸投げ。

## 2. ファイル読み書き(30分)

### テキスト

```python
# 書き込み
with open("memo.txt", "w", encoding="utf-8") as f:
    f.write("こんにちは\n")
    f.write("2行目")

# 読み込み
with open("memo.txt", "r", encoding="utf-8") as f:
    内容 = f.read()
    print(内容)

# 行ごと
with open("memo.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

| モード | 用途 |
|---|---|
| `"w"` | 書き込み(上書き) |
| `"r"` | 読み込み |
| `"a"` | 追記 |

`encoding="utf-8"` は日本語必須。

### JSON

```python
import json

data = {"名前": "太郎", "趣味": ["将棋", "読書"]}

# 書き込み
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 読み込み
with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)
print(data["名前"])
```

### CSV

```python
import csv

# 書き込み
データ = [
    {"名前": "太郎", "年齢": 25},
    {"名前": "花子", "年齢": 30},
]
with open("data.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=["名前", "年齢"])
    writer.writeheader()
    writer.writerows(データ)

# 読み込み
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row["名前"], row["年齢"])
```

→ ただし**CSV/Excelは次章のpandasがメイン**。

## 3. フォルダ操作(20分)

```python
from pathlib import Path

p = Path(".")

# 中身一覧
for f in p.iterdir():
    print(f.name)

# 拡張子で絞る
for f in p.glob("*.csv"):
    print(f.name)

# 再帰的(サブフォルダも)
for f in p.rglob("*.py"):
    print(f.name)

# 存在確認
Path("memo.txt").exists()

# フォルダ作成
Path("新しいフォルダ").mkdir(exist_ok=True)
```
