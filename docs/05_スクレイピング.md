## 2. 基本(20分)

```python
import requests

r = requests.get("https://example.com")
print(r.status_code)  # 200=成功
print(r.text[:500])    # HTML最初500文字
```

### ステータスコード

| コード | 意味 |
|---|---|
| 200 | 成功 |
| 403 | アクセス拒否 |
| 404 | ページなし |
| 429 | アクセスしすぎ |

### User-Agent(ブラウザのフリ)

```python
headers = {
    "User-Agent": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)"
}
r = requests.get(url, headers=headers)
```

## 3. BeautifulSoup で抽出(30分)

```python
from bs4 import BeautifulSoup

html = """
<div class="article">
  <h2 class="title">記事タイトル</h2>
  <p class="body">本文</p>
</div>
"""

soup = BeautifulSoup(html, "lxml")

# タグで取得
soup.find("h2").text                       # "記事タイトル"
soup.find("div", class_="article")          # divごと

# CSSセレクタ
soup.select_one(".title").text             # "記事タイトル"
soup.select(".article p")                  # pタグ全部

# 全リンク
for a in soup.find_all("a"):
    print(a.get("href"))
```

### CSSセレクタ早見表

| 記号 | 意味 |
|---|---|
| `.class` | クラス |
| `#id` | ID |
| `tag` | タグ名 |
| `親 > 子` | 直接の子 |
| `親 子孫` | 全子孫 |

## 5. API を使う(API があれば優先)(30分)

API = サイトが提供する**公式の窓口**。スクレイピングより**100倍楽**。

### GitHub API例

```python
import requests

r = requests.get("https://api.github.com/users/octocat")
data = r.json()  # JSONを辞書に
print(data["name"])  # "The Octocat"
```

### 為替レートAPI(無料)

```python
r = requests.get("https://open.er-api.com/v6/latest/USD")
data = r.json()
jpy = data["rates"]["JPY"]
print(f"1 USD = {jpy} JPY")
```

## 6. JavaScript で動的に読み込まれるサイト(参考)

普通の `requests` だと取れない場合は **Playwright**:

```
pip install playwright
playwright install chromium
```

```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto("https://example.com")
    page.wait_for_load_state("networkidle")
    html = page.content()
    browser.close()
```

→ JS実行後のHTMLが取れる。**慣れたら使う**。

## 7. ⭐ 売れる成果物 #2:価格収集ツール(60分)

### 仕様

- 商品名のリストを受け取り
- 各商品をECサイト(規約OKな範囲で)で検索
- 価格・在庫・URL を取得
- CSVに保存

### クロードに頼む

```
以下を実装するPythonコード書いて。

価格比較.com のような価格情報サイト(または規約上問題ないサイト)で
指定キーワードを検索して、商品名・価格・URLを取得して
CSVに保存するスクリプト。

要件:
- requests + BeautifulSoup
- User-Agent指定
- 1秒間隔
- CSVは utf-8、1列目「商品名」、2列目「価格」、3列目「URL」
- 取得失敗時のエラーハンドリング
- 進捗表示

サンプル: 「Switch」「PS5」「Xbox」 を検索して、各上位5件取得。

※スクレイピング対象URLは規約上問題ないサイトを選ぶ提案も含めて。
```

### 別案:API使う

```
為替レートAPI (https://open.er-api.com/v6/latest/USD) と
GitHub APIを使って、各レポジトリの星数と現在のドル円を毎日取得し、
CSVに追記していくスクリプトを書いて。
```
