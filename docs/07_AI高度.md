## 1. RAG とは(15分)

「**Retrieval Augmented Generation**」 = 検索 + AI生成

```
[PDF/Word/CSV] → 細かく分割 → ベクトル化(意味を数字に) → DB保存
   ↓
ユーザー質問 → ベクトル化 → DBから関連文書TOP3検索 → AIに渡す → 回答
```

### なぜ必要?

ChatGPT/Claude は**社内情報を知らない**。RAGで:
- 自社のPDF50枚を読ませる
- 「就業規則の有給はどう?」と聞ける
- AIが**自社情報に基づいて**回答

→ 法律事務所・士業・医療機関が**月50-200万円で発注**。

## 4. LangChain Expression Language(LCEL)(30分)

LangChain の最新書き方。**チェーン処理が楽**。

```python
from langchain_anthropic import ChatAnthropic
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

llm = ChatAnthropic(model="claude-opus-4-7")
prompt = ChatPromptTemplate.from_messages([
    ("system", "あなたは丁寧な秘書です"),
    ("human", "{question}"),
])

chain = prompt | llm | StrOutputParser()
print(chain.invoke({"question": "明日の段取りを教えて"}))
```

→ プロンプト → AI → 出力 をパイプで繋ぐ。
