# Day 11:ランサーズ・直接契約 - 即金収入

## 💡 これは何?(小学生でもわかる説明)

**仕事を探す場所(ランサーズ)に「自分の名刺」を置いて、お客さんから仕事をもらう日**

### なぜ大事?
ここで初受注 → 月3〜30万円スタート。
**0円エンジニアと月収1万エンジニアの間には深い壁がある**。
1件でも納品して★を貰えば、その後は自動で次の依頼が来やすくなる。

### イメージ
- **ランサーズ**=「**お仕事ハローワーク**」(仕事募集の掲示板)
- **プロフィール**=「**お店の看板**」(これが弱いと誰も入って来ない)
- **提案文**=「**ラブレター**」(他の応募者と差をつける一通の手紙)
- **★5評価**=「**Googleの星マーク**」(これが貯まると客が勝手に来る)

### Claude Code に何を頼めばいいか
プロフィール文・提案文・スキルシート・納品メッセージは**全部クロードに書かせる**。
あなたは「**素材を集める**」「**送信ボタンを押す**」だけ。

## 1. 3サイト登録(=お店を3軒オープン)(40分)

### 1-1. なぜ3サイト?

> **1サイトだけだと魚の数が少ない。3つに釣り糸を垂らしておけば、どれかに必ず食いつく**。

各サイトで案件の傾向が違う:
- ランサーズ → 個人事業主が多い、案件量No.1
- クラウドワークス → 法人案件が比較的多い
- ココナラ → こちらから応募しなくても客が来る(出品型)

### 1-2. 必須登録

| サイト | URL | 特徴 | 案件の色 |
|---|---|---|---|
| **ランサーズ** | lancers.jp | 案件量No.1 | 自動化・スクレイピング多い |
| **クラウドワークス** | crowdworks.jp | ランサーズと両刀 | データ入力・記事系も多い |
| **ココナラ** | coconala.com | パッケージ売り、待ちでも案件来る | 「〇〇します」型出品 |

3つ全部登録。**本人確認**まで完了させる(運転免許証 or マイナンバー)。

### 1-3. 本人確認は絶対やる

> **本人確認済みのバッジは「信頼度3倍」のお守り**。

クライアント側からすると「**身元のわからん人にお金は払えん**」。
免許証を1枚アップするだけで、**スカウト率が体感で3倍**になる。

### よくある間違い・引っかかりポイント

| やりがちなミス | 何がダメ | 直し方 |
|---|---|---|
| アイコンがデフォルト | 「やる気あんの?」と思われる | 顔写真 or イラストを必ず設定 |
| 自己紹介が3行 | スカスカで信頼ゼロ | 後の章で1500字書く |
| メアドだけ登録、本人確認スキップ | 高単価案件は応募できない | 免許証で本人確認まで完了 |

## 2. プロフィール文(=お店の看板)(40分)

### 2-1. なぜ1500字も書く?

> **クライアントは応募してきた10人のプロフィールを見比べて1人を選ぶ**。
> 3行のプロフィールは「**他より自信なさそう**」と判定される。
> 1500字あれば「**ちゃんと考えてる人**」だと思われる。

### 2-2. クロードに頼むテンプレ

```
ランサーズのプロフィール文を1500字で書いて。

私の特徴:
- Pythonでの業務自動化が得意
- データ分析・スクレイピング経験
- Claude API / OpenAI API でAIアプリ構築可能
- ポートフォリオ: <GitHub URL>

提供サービス:
- Excel/CSV/PDF処理(5,000-30,000円)
- スクレイピング(10,000-50,000円)
- データ分析・レポート(20,000-100,000円)
- AIアプリ開発(50,000-300,000円)

対応時間: 平日19-23時、土日終日

トーン: 丁寧 + 具体的 + 信頼感
```

→ クロードが書いてくれた文を**3サイトにコピペ**。

### 2-3. プロフィールに必ず入れる5要素

1. **挨拶 + 何屋さんか**(冒頭で「Pythonで業務自動化やってます」)
2. **得意分野**(箇条書きで5個くらい)
3. **過去実績 or ポートフォリオURL**(GitHubリンク必須)
4. **対応時間 + 連絡レスポンス**(「24時間以内に必ず返信」)
5. **料金の目安**(これがあると交渉が早い)

### 2-4. スキルタグ(検索でヒットさせる呪文)

全部入れる:
- Python
- データ分析
- 業務自動化
- スクレイピング
- 機械学習
- AI / ChatGPT API / Claude API
- Excel / pandas

> **なぜタグ?**
> クライアントは検索ボックスに「Python AI」と打って人を探す。
> タグが入ってないと**検索結果に出ない=存在してないのと同じ**。

## 3. 案件の選び方(=毒キノコを避ける)(20分)

### 3-1. なぜ選ぶ必要がある?

> **応募する前に「この案件は地雷じゃないか」を見抜く**。
> 地雷案件を1つ受けると、**1ヶ月分の時間とメンタルが死ぬ**。

### 3-2. 検索ワード

- 「Python」
- 「Excel 自動化」
- 「スクレイピング」
- 「ChatGPT API」「Claude」
- 「データ分析」

### 3-3. 良い案件 vs 悪い案件の見分け方

| 観点 | 良い | 悪い |
|---|---|---|
| 仕様 | 明確 | あいまい(「いい感じに」「適当に」) |
| 予算 | 5,000-30,000円 | 1,000円以下 |
| クライアント | 評価★4以上、認証済み | 評価なし、未認証 |
| 期間 | 1週間以上 | 「今日中」 |
| 応募数 | 5-20件 | 50件超(競争激しすぎ) |

### 3-4. 避けるべき罠(=詐欺フィルター)

- 「コピペ作業で月10万!」 → ほぼ詐欺(マルチ商法・情報商材)
- 「LINE/メールで連絡先教えて」 → **規約違反**(後で踏み倒される)
- 完全成果報酬 → 払われないリスク高い
- 「ノウハウ料5万円先払い」 → 詐欺確定、即通報

> **判断基準**: 「**プロフィールに事業実態が書いてあるか**」を見る。
> 個人名のみ・実績ゼロのクライアントは要注意。

## 4. 提案文の書き方(=ラブレターの書き方)(60分)

### 4-1. なぜ提案文が大事?

> **クライアントは20通のラブレター(提案)を読み比べる**。
> 「やります」だけの3行ラブレターは、開封すらされない。
> **「あなたの悩みを理解した」「私はこう解決する」**を書いた手紙だけが選ばれる。

### 4-2. NG提案(やりがち)

```
こんにちは。やります。3日でできます。
```
→ 落選確定。**何も伝わらない**。

### 4-3. OK テンプレ(全部入り)

```
〇〇様

はじめまして、Pythonエンジニアの△△と申します。
ご依頼の【XXX】について提案させていただきます。

▼ 解決方法
1. Pythonスクリプト(pandas使用)
2. <具体的な実装イメージ>
3. <納品物の形式>

▼ 想定スケジュール
- ヒアリング: 1日
- 実装: 3日
- テスト・修正: 1日
合計 5営業日

▼ 想定価格
〇〇円(調整可)

▼ 過去実績(GitHub)
- https://github.com/xxx/python-portfolio
  → 類似のExcel自動化ツール

▼ 確認事項(返信不要)
- ファイル数の想定(10〜100件?)
- 出力形式(Excel/CSV?)

ご不明点あればお気軽にお問い合わせください。
よろしくお願いいたします。

△△
```

> **このテンプレが採用される理由:**
> - 名乗り → 安心感
> - 解決方法 → 「ちゃんと考えてる」感
> - スケジュール → プロっぽい
> - 価格 → 交渉前提で調整可
> - 実績 → 嘘つきじゃない証拠
> - 確認事項 → クライアントの仕様を深掘り済み

### 4-4. クロードに頼むテンプレ

```
以下のランサーズ案件への提案文を書いて。

【案件文】
<ここにコピペ>

【私の状況】
- Python自動化得意
- ポートフォリオ: <URL>
- 単価希望: 〇〇円
- 期間: △日

トーン: 丁寧 + プロフェッショナル
800-1200字
```

→ クロード生成 → 微調整 → 提出。

### 4-5. やりがちな失敗

| ミス | 何がダメ | 直し方 |
|---|---|---|
| 全案件にコピペ提案 | 案件番号や担当者名が違う、即バレる | 案件名・依頼内容を提案文に1行入れる |
| 自分語りばかり | クライアントは「自分の問題が解けるか」しか興味ない | 80%は「あなたの問題をこう解く」 |
| 単価を書かない | 比較できないのでスルーされる | 「〇円(調整可)」と必ず書く |

## 5. ⭐ 初回戦略:単価半額(=実績作りの捨て駒)(20分)

### 5-1. なぜ半額にする?

> **評価ゼロのうちは「あなたを選ぶ理由が他人より弱い」**。
> 価格を下げてストーリー(=ランサーズデビュー記念)を付ければ、**選ばれる確率が3倍**になる。

### 5-2. 具体的な値付け

評価ゼロのうちは:
- 通常 10,000円 → **5,000円**で提案
- 通常 30,000円 → **15,000円**

提案文にこう書く:
```
※ランサーズデビュー記念として通常〇〇円のところ△△円にて
対応させていただきます。
```

→ ストーリー性で**採用率3倍**。

### 5-3. 評価★5を3-5件貯めるのが先

最初の3-5件は**実績作り**と割り切る。
それ以降は通常単価。**最初の半年で年収勝負しない**。

> **長期目線:**
> 半額で5件こなす(=2万円)→ 通常単価で月20万に化ける。
> ここをケチると永遠に時給500円ループ。

## 6. 5件応募(=毎日の打席に立つ)(40分)

### 6-1. なぜ5件?

> **応募1件で受注できる確率は5〜10%**。
> 1件しか応募しなければ、**当然1件も受注しない**。
> 5件出して、1件返事が来れば上出来。

### 6-2. 進捗管理表

毎日5件、新着案件を見る。
今日中に**5件応募する**(クロードに提案文書かせれば30分)。

| 案件 | サイト | 単価 | 状態 |
|---|---|---|---|
| Excel統合 A社 | ランサーズ | 8,000円 | 提案中 |
| スクレイピング B社 | クラウドワークス | 15,000円 | 採用 |
| データ整形 C社 | ランサーズ | 5,000円 | 不採用 |
| ... | ... | ... | ... |

→ Notion or Excel で管理。**応募した日付**も記録。

### 6-3. 続けるコツ

- 平日朝の通勤前に新着チェック(5分)
- 昼休みに気になる案件をブックマーク
- 夜21時にクロードに提案文書かせて送信

> **半年で「ランサーズだけで月20万」のラインに乗る**。

## 8. ⭐ SES エージェント登録(=もう1段上の収入源)(40分)

### 8-1. SESって何?

ランサーズ単発と並行して、**SES(準フリーランス契約)** に登録すると単価爆上がり。

> **SES**=「**準フリーランス契約**」
> エージェント経由で企業の開発チームに参加する形態。
> イメージは「**派遣社員のフリーランス版**」。

### 8-2. なぜSESが強い?

| 比較項目 | ランサーズ単発 | SES |
|---|---|---|
| 単価 | 1件5,000〜30,000円 | 月60-100万円 |
| 安定性 | 毎月新規営業必要 | 月単位で契約継続 |
| スキル要件 | 入門〜中級 | 中級以上(でもOJTあり) |

> **SES 1社で月70万**を達成すると、**ランサーズ毎日5件応募**から解放される。

### 8-3. 主要エージェント

| エージェント | 特徴 | 単価レンジ |
|---|---|---:|
| **レバテックフリーランス** | 案件量No.1 | 60-100万/月 |
| **Midworks** | 若手向け、保証あり | 60-90万/月 |
| **ITプロパートナーズ** | 週2-3日案件多い | 50-80万/月 |
| **PE-BANK** | マージン低い | 60-100万/月 |

### 8-4. 登録手順

1. オンライン登録(15分)
2. スキルシート提出(クロードに作らせる)
3. エージェントと面談(60分、Web)
4. 案件紹介(数日)
5. 企業面談 → 契約

→ ★5評価3件 + ポートフォリオがあれば**1ヶ月で案件決まる**。

### 8-5. スキルシートのテンプレ

クロードに頼む:

```
SES向けスキルシートを書いて。

【経験】
- Python (1年、独学+案件)
- pandas / requests / BeautifulSoup
- LangChain / RAG構築
- Streamlit / FastAPI
- Git / GitHub

【ポートフォリオ】
- <5作品のリスト>

【希望】
- 単価: 60-80万/月
- 稼働: 週3-5日
- リモート希望

A4 2ページに収まる量。
```

→ これで月60-80万円圏内のSES案件を狙える。

### 8-6. SES面談で聞かれる定番質問

| 質問 | 回答テンプレ |
|---|---|
| なぜフリーランスに? | 「自分のスキルで価値提供したい + 自由な働き方を選びたい」 |
| 得意領域は? | 「Pythonでの業務自動化とAI連携」(具体例も) |
| 月の稼働時間? | 「週20時間〜40時間で柔軟に」 |
| 何ヶ月の契約希望? | 「3ヶ月〜6ヶ月、相互に合えば延長」 |

→ クロードに「SES面談の想定問答10個書いて」で完璧版もらえる。
