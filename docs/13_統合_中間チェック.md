## 2. SES で月60-80万円(60分)

### SESとは

「**準フリーランス契約**」。エージェント経由で企業と直接契約。

- 週3-5日、企業の開発チームに参加
- リモート可
- 月単価 60-100万円

### 主要エージェント

| エージェント | 平均単価 | 特徴 |
|---|---:|---|
| **レバテックフリーランス** | 80万 | 案件量No.1 |
| **Midworks** | 70万 | 保証あり |
| **ITプロパートナーズ** | 65万 | 週2-3日案件多い |
| **PE-BANK** | 90万 | マージン低い |

### 単価交渉のコツ

1. **複数エージェント登録**(競争させる)
2. **スキルシートに数字**(削減時間、改善%)
3. **「他社で◯◯万」と提示**(嘘はNG)
4. **試用期間1ヶ月で評価UP→更新時に交渉**

### 注意点

- **常駐型より準委任型を選ぶ**(自由度高い)
- **副業禁止条項に注意**(月額契約と両立できる契約)
- **稼働終了後の継続率**を必ず確認

→ **SES 1社で月70万 + 月額契約3社で月20万 = 月90万** が王道。

## 4. 法人化の判断(20分)

### いつ法人化?

- 年売上 800-1,000万円超
- 月収50万安定
- 経費が多い(年300万超)

### 法人化のメリット

- 税率(個人最大55% vs 法人約30%)
- 経費の幅が広がる
- 信頼性UP(月100万案件取りやすい)
- 退職金積立可能

### デメリット

- 設立費用 20-30万円

- 決算手続き(税理士月3-5万円)

→ 月100万安定したら**税理士に相談 → 法人化検討**。
