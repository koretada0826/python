## 5. SES で月60-80万円(=2階の柱)(60分)

### 5-1. SESとは

「**準フリーランス契約**」。エージェント経由で企業と直接契約。

- 週3-5日、企業の開発チームに参加
- リモート可
- 月単価 60-100万円

### 5-2. 主要エージェント

| エージェント | 平均単価 | 特徴 |
|---|---:|---|
| **レバテックフリーランス** | 80万 | 案件量No.1 |
| **Midworks** | 70万 | 保証あり |
| **ITプロパートナーズ** | 65万 | 週2-3日案件多い |
| **PE-BANK** | 90万 | マージン低い |

### 5-3. 単価交渉のコツ

1. **複数エージェント登録**(競争させる)
2. **スキルシートに数字**(削減時間、改善%)
3. **「他社で◯◯万」と提示**(嘘はNG)
4. **試用期間1ヶ月で評価UP→更新時に交渉**

### 5-4. SESの注意点

- **常駐型より準委任型を選ぶ**(自由度高い)
- **副業禁止条項に注意**(月額契約と両立できる契約)
- **稼働終了後の継続率**を必ず確認

> **王道:** SES 1社で月70万 + 月額契約3社で月20万 = **月90万**。

## 6. SaaS / 自分のサービス積み上げ(=屋上の小屋)(60分)

### 6-1. なぜSaaSが屋上?

> **SaaS=「自動で増える収入源」**。
> ユーザーが100人増えれば月10万、300人で月30万。
> **時間と関係なく増える**ので、月額・SESと違って「**働かない時間にも収入が増える**」。

### 6-2. Day 8 で作ったSaaSを伸ばす

#### マイルストーン

| 期間 | ユーザー数 | MRR(月収) |
|---|---:|---:|
| 1ヶ月 | 5 | 5,000円 |
| 3ヶ月 | 30 | 30,000円 |
| 6ヶ月 | 100 | 100,000円 |
| 12ヶ月 | 300 | 300,000円 |

→ **2年で月30万のSaaS収入**は再現性ある。

### 6-3. マーケティング戦略(=種を撒く5つの場所)

1. **Twitter継続発信**(Day 10-13で半年-1年)
2. **Note 記事**(SEOで月10記事)
3. **ProductHunt 投稿**(英語・初日にバズ狙い)
4. **YouTube 解説動画**(機能紹介)
5. **Reddit / HackerNews**(英語SaaSなら効く)

### 6-4. 価格戦略(Freemium 推奨)

```
無料: 月10回まで使える
有料: 1,000円/月(無制限) ← メイン
プロ: 5,000円/月(高度機能 + 法人サポート)
法人: 50,000円/月(社内利用無制限)
```

→ **法人プランが効く**(1法人 = 個人50人分)

> **Freemium の心理:** 無料で使ってもらう → 「いいね」と思ったら自然に有料へ → 解約しにくくなる。

## 7. 法人化の判断(=会社を作るかどうか)(20分)

### 7-1. なぜ法人化?

> **個人で月100万を超えると、税率が55%まで上がる**(=半分が税金)。
> 法人にすると約30%。**手取りが100万円違う年もある**。
> ただし設立費用とランニングコストがかかるので、**月収50万安定してから検討**。

### 7-2. いつ法人化?

- 年売上 800-1,000万円超
- 月収50万安定
- 経費が多い(年300万超)

### 7-3. 法人化のメリット・デメリット

| メリット | デメリット |
|---|---|
| 税率(個人最大55% vs 法人約30%) | 設立費用 20-30万円 |
| 経費の幅が広がる | 社会保険加入(月7-15万円) |
| 信頼性UP(月100万案件取りやすい) | 決算手続き(税理士月3-5万円) |
| 退職金積立可能 | 赤字でも法人住民税 約7万円/年 |

→ 月100万安定したら**税理士に相談 → 法人化検討**。

## 8. リスク管理(=保険をかける)(20分)

### 8-1. 月100万を**失う3大リスク**

#### リスク1. クライアント1社依存
**対策**: 売上の25%超を1社に依存しない

> **理由:** 1社の解約で収入半減 → 心が折れて全部辞める人多い。

#### リスク2. 単一スキル依存
**対策**: AI / Web / データ の複数領域

> **理由:** 「AIだけ」だと、AIブームが終わった瞬間ゼロ。

#### リスク3. 健康問題
**対策**: 月10時間以上の稼働増は断る、週1日完全休み

> **理由:** 稼働しすぎ → 体壊す → 全部止まる。
> 月100万維持は**稼ぎ方より休み方**で決まる。

### 8-2. キャッシュ管理

```
個人口座: 6ヶ月分の生活費を死守
事業口座: 経費・税金分を別管理
投資口座: 余剰金は積立NISA等
```

→ **半年休んでも生きられる貯金**が心の平穏。
