## 1. 業務委託契約書(40分)

### 必ずチェックする10項目

1. **業務範囲**(何までやる?)
2. **納期**(いつまで?)
3. **報酬**(いくら、いつ振込?)
4. **支払い条件**(月末締め翌月末払い等)
5. **検収条件**(何を満たせば完了?)
6. **修正・追加対応**(何回まで無料?)
7. **知的財産権**(成果物の権利は誰?)
8. **秘密保持**(NDA別契約?契約書内?)
9. **損害賠償**(上限額)
10. **解除条件**(どんな時解約できる?)

### 危険な条項

❌ **「全ての知的財産権はクライアントに無償譲渡」**
→ 自分のノウハウまで奪われる、譲渡範囲明確化

❌ **「納品物に関する一切の不具合の責任を負う」**
→ 賠償上限なし、無限責任は危険

❌ **「無限の修正対応」**
→ 「契約金額の範囲内で〇回まで」に修正

### 修正交渉

```
契約書の確認をさせていただきました。
以下の点について修正をお願いできますでしょうか?

1. 7条(知的財産権)
  → 「本件業務に関する成果物の著作権」と限定したい
  → 弊社の汎用ライブラリは含めない

2. 9条(損害賠償)
  → 「契約金額を上限とする」を追記したい

トラブル防止の観点での確認です。
よろしくお願いします。
```

### 自分から契約書を出す

直接契約時。クロードに頼む:

```
業務委託契約書のテンプレを書いて。

仕事内容: Pythonによる業務自動化開発
期間: 単発(納品ベース)
報酬: 〇〇円(税込み)
支払い: 検収後14日以内
納期: 契約日から△日
著作権: 成果物の著作権は乙(私)に帰属、利用権を甲に許諾
損害賠償: 契約金額を上限
解除条件: 双方の合意 or 重大な違反

これでドラフト作成。
```

→ クラウドサインで電子契約締結。

## 2. NDA(秘密保持契約)(20分)

法人案件で**ほぼ必ず**出てくる。

### 主要チェック

- **秘密情報の定義**(何が機密?)
- **有効期間**(契約終了後何年?)
- **目的外利用禁止**
- **損害賠償**

### 注意すべき条項

❌ **「全ての情報を機密扱い」**
→ 公知情報・自分が既に知ってた情報は除外させる

❌ **「永久に秘密保持」**
→ 通常3-5年で十分

### 自分用 NDA テンプレ

クロードに「フリーランス Python エンジニア用のNDA テンプレを書いて」で生成。

## 3. インボイス制度(2023年〜)(20分)

### インボイス登録すべき?

#### YES(登録した方がいい)
- クライアントの大半が**法人(課税事業者)**
- 単価交渉で「インボイスなしだと10%引きで」と言われる
- 売上1,000万円超(自動的に課税事業者)

#### NO(登録しない方がいい)
- クライアントが**個人・免税事業者**中心
- 売上 1,000万円未満で当面増えない

### 登録のデメリット

- 消費税納税義務(売上の10%)
- 簡易課税で**みなし仕入率50%**(IT業)→ 実質売上の5%納税

→ クライアントに**法人多い人は登録**、個人相手は様子見。

### 確認方法

「**Tから始まる13桁の番号(T1234567890123)を発行**」

## 4. 確定申告・節税(40分)

### 個人事業主の手順

1. **開業届**(税務署、無料、5分)
2. **青色申告承認申請書**(開業届と同時)
3. **freee/マネーフォワード**契約(月980円〜)
4. レシート全部保存
5. 翌年2-3月に確定申告

### 青色申告の特典

- **65万円控除**(電子申告)
- 赤字を3年繰越可
- 家族への給与を経費に
- 30万円未満の備品を一括経費

### 経費にできるもの(全部入れる)

| 区分 | 例 |
|---|---|
| 通信費 | ネット代、電話代、SaaS月額 |
| 消耗品 | 文房具、PCパーツ |
| 旅費交通費 | 取引先訪問、勉強会 |
| 接待交際費 | クライアント飲食 |
| 図書 | 技術書、講座、Audible |
| 広告宣伝費 | 名刺、ホスティング |
| 研修費 | Udemy、コース、コンサル |
| 減価償却 | PC(10万超は分割) |
| 地代家賃 | 自宅家賃の30-50%(家事按分) |
| 水道光熱費 | 自宅光熱費の30%程度 |
| 租税公課 | 個人事業税、印紙税 |

→ **月収100万 ÷ 経費20万 → 課税所得80万**(節税効果月8万)

### 節税の上級技

#### 1. 小規模企業共済(年84万円控除)
- 個人事業主向け退職金制度
- 全額所得控除
- いつでも解約可

#### 2. iDeCo(年27.6万円控除)
- 60歳まで引き出せないが**節税効果絶大**

#### 3. ふるさと納税(実質負担2,000円)
- 年収1,000万 → 約17万円分の返礼品

#### 4. 法人化(年商800-1,000万円超で検討)
- 法人税率 23%(個人最大45%より低い)
- 経費の幅広がる
- 退職金積立可能
