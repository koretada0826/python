## 3. リスク管理(15分)

### 3-1. 月100万を**失う5つのリスク**と対策

| リスク | 何が起きる | 対策 |
|---|---|---|
| 1社依存 | その1社が切られたら半分以上消滅 | 売上25%超を1社に依存しない |
| スキル古化 | 3年で技術が陳腐化 | 月10時間の学習継続 |
| 健康問題 | 倒れたら収入0 | 週1日完全休み・健康診断 |
| 税金問題 | 数百万の追徴課税 | 税理士月3-5万円契約 |
| クライアント倒産 | 売掛金回収不能 | 取引前に与信チェック・契約書必須 |

### 3-2. キャッシュ管理

```
個人口座: 6ヶ月分の生活費(死守)
事業口座: 経費・税金分(別管理)
投資口座: 余剰金は積立NISA・iDeCo
```

> **なぜ6ヶ月分の生活費?**
> 主軸クライアントが切られても、半年あれば次を取れる。
> 余裕があると「足元見られた値下げ交渉」を断れる。

### 6-1. 月100万到達後の罠

| 罠 | 起きること | 防ぎ方 |
