## 2. データ構造の使い分け(40分)

### list vs deque vs heapq

```python
# list: ランダムアクセスO(1)、末尾追加O(1)、先頭追加O(n)
arr = [1,2,3]
arr.append(4)   # O(1)
arr.insert(0,0) # O(n) ← 遅い

# deque: 両端O(1)
from collections import deque
dq = deque([1,2,3])
dq.appendleft(0)  # O(1)
dq.popleft()      # O(1)

# heapq: 最小値取得O(log n)
import heapq
h = []
heapq.heappush(h, 3)
heapq.heappush(h, 1)
heapq.heappush(h, 2)
heapq.heappop(h)  # 1
```

### dict / set(ハッシュテーブル)

```python
# 検索O(1)
d = {"a": 1, "b": 2}
"a" in d  # O(1)

s = {1, 2, 3}
2 in s    # O(1)
```

→ **「リスト in 検索」O(n)** を **「set in 検索」O(1)** に置き換えるだけで**100倍速**。

## 3. 主要アルゴリズム(60分)

### 二分探索 O(log n)

```python
def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target: return mid
        if arr[mid] < target: lo = mid + 1
        else: hi = mid - 1
    return -1

# 標準ライブラリ
import bisect
bisect.bisect_left(arr, target)
```

### ソート

```python
arr.sort()                     # 安定ソート O(n log n)

```

### 再帰

```python
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)

# メモ化で高速化
from functools import cache
@cache
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)
```

### DP(動的計画法)

「**重複する部分問題**を表で記録**」。

```python
# 階段問題:n段の階段、1段または2段ずつ。何通り?
def climb(n):
    dp = [0] * (n+1)
    dp[0] = dp[1] = 1
    for i in range(2, n+1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

### BFS / DFS(グラフ探索)

```python
# BFS(幅優先)— 最短経路
from collections import deque
def bfs(graph, start):
    visited = {start}
    q = deque([start])
    while q:
        node = q.popleft()
        print(node)
        for next_node in graph[node]:
            if next_node not in visited:
                visited.add(next_node)
                q.append(next_node)

# DFS(深さ優先)— 全経路
def dfs(graph, node, visited=None):
    if visited is None: visited = set()
    visited.add(node)
    print(node)
    for next_node in graph[node]:
        if next_node not in visited:
            dfs(graph, next_node, visited)
```

### Two Pointers

```python
# ソート済み配列で和がtargetのペアを探す O(n)
def two_sum(arr, target):
    l, r = 0, len(arr)-1
    while l < r:
        s = arr[l] + arr[r]
        if s == target: return (l, r)
        if s < target: l += 1
        else: r -= 1
```

### Sliding Window

```python
# 連続するK個の最大和
def max_sum_k(arr, k):
    s = sum(arr[:k])
    best = s
    for i in range(k, len(arr)):
        s += arr[i] - arr[i-k]
        best = max(best, s)
    return best
```

## 4. 木とグラフ(40分)

### 二分木

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# 中順走査(左→根→右)
def inorder(node):
    if not node: return
    inorder(node.left)
    print(node.val)
    inorder(node.right)
```

### グラフ表現

```python
# 隣接リスト(おすすめ)
graph = {
    "A": ["B", "C"],
    "B": ["A", "D"],
    "C": ["A", "D"],
    "D": ["B", "C"],
}
```

### Dijkstra(最短経路)

```python
import heapq
def dijkstra(graph, start):
    dist = {n: float("inf") for n in graph}
    dist[start] = 0
    pq = [(0, start)]
    while pq:
        d, node = heapq.heappop(pq)
        if d > dist[node]: continue
        for next_node, w in graph[node]:
            nd = d + w
            if nd < dist[next_node]:
                dist[next_node] = nd
                heapq.heappush(pq, (nd, next_node))
    return dist
```
