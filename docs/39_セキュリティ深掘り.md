## 1. JWT 完全理解(30分)

### 構造

```
xxxxx.yyyyy.zzzzz
↑      ↑      ↑
Header Payload Signature
```

```python
import jwt
SECRET = "your-secret"

# 発行
token = jwt.encode({
    "sub": "user123",
    "exp": datetime.utcnow() + timedelta(hours=1)
}, SECRET, algorithm="HS256")

# 検証
payload = jwt.decode(token, SECRET, algorithms=["HS256"])
```

### よくある脆弱性

❌ **alg: none** を許可 → 署名なしで認証通過
❌ **HS256** で SECRET を弱い文字列に
❌ JWT を **localStorage** に置く → XSSで盗まれる
✅ **httpOnly Cookie** に置く

## 2. OAuth2 / OIDC(30分)

「Google でログイン」の仕組み。

```
[ユーザー] →[アプリ] → 「Googleにアクセス権ください」
              ↓
[Google] ←──── ログイン → 同意画面 → アクセストークン発行
              ↓
[アプリ] ← トークンで API 呼び出し → ユーザー情報取得
```

### Authlib で実装

```bash
pip install authlib
```

```python
from authlib.integrations.starlette_client import OAuth

oauth = OAuth()
oauth.register(
    name="google",
    client_id=os.getenv("GOOGLE_CLIENT_ID"),
    client_secret=os.getenv("GOOGLE_SECRET"),
    server_metadata_url="https://accounts.google.com/.well-known/openid-configuration",
    client_kwargs={"scope": "openid email profile"}
)
```

→ 自社認証作るより**Google任せ**が安全・楽。

## 3. 暗号(15分)

### ハッシュ(一方向)

```python
# ❌ MD5/SHA1: 古い、脆弱
# ✅ bcrypt / argon2 / scrypt
from passlib.hash import bcrypt
hash = bcrypt.hash("password123")
bcrypt.verify("password123", hash)
```

### 対称鍵暗号(双方向、同じ鍵)

```python
from cryptography.fernet import Fernet
key = Fernet.generate_key()
f = Fernet(key)
enc = f.encrypt(b"secret")
plain = f.decrypt(enc)
```

### 公開鍵暗号

```bash
ssh-keygen -t ed25519
# 公開鍵で暗号化 → 秘密鍵でしか復号できない
```

→ SSH/SSL/署名 で使われる。

## 4. CSRF / SSRF / XXE(15分)

### CSRF(他サイトから自分のAPIを呼ばれる)

防御:
- **SameSite Cookie**
- **CSRF Token**

### SSRF(サーバーから内部攻撃)

```python
# ❌ ユーザー入力URLをそのまま fetch
url = request.json["url"]
requests.get(url)  # → http://169.254.169.254 でAWSメタデータ取得攻撃

# ✅ ホワイトリスト
ALLOWED_HOSTS = ["api.example.com"]
```

### XXE(XML 外部エンティティ)

```python
# 古い XML パーサー → 攻撃可能
import defusedxml.ElementTree as ET  # ✅ 安全版
```
