## 2. Unit Test(基礎)(20分)

Day 18でカバー済み。pytest 復習。

```python
def test_add():
    assert add(2, 3) == 5

@pytest.mark.parametrize("a,b,expected", [(2,3,5), (-1,1,0)])
def test_add_param(a, b, expected):
    assert add(a, b) == expected
```

## 4. E2E Test(Playwright)(30分)

```bash
pip install pytest-playwright
playwright install
```

```python
def test_login(page):

    page.fill("#email", "x@y.z")

    page.click("button[type=submit]")
```

## 5. Property-Based Testing(20分)

```bash
pip install hypothesis
```

```python
from hypothesis import given, strategies as st

@given(st.lists(st.integers()))
def test_sort_idempotent(xs):
    assert sorted(sorted(xs)) == sorted(xs)

@given(st.text())
def test_encode_decode(s):
    assert decode(encode(s)) == s
```

→ **何百パターン**を自動生成してくれる。バグを見つける確率高い。

## 6. Mocking 高度(15分)

```python
from unittest.mock import patch, MagicMock

# 外部API呼び出しをモック
@patch("app.requests.get")
def test_fetch(mock_get):
    mock_get.return_value.json.return_value = {"data": [1,2,3]}
    result = fetch_users()
    assert result == [1,2,3]
    mock_get.assert_called_once_with("https://api.example.com/users")
```

## 8. CI でテスト自動化(15分)

`.github/workflows/test.yml`:
```yaml
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    services:
      postgres:
        image: postgres:16
        env:
          POSTGRES_PASSWORD: postgres
        ports: [5432:5432]
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with: { python-version: "3.12" }
      - run: pip install -r requirements.txt
      - run: pytest --cov=. --cov-fail-under=80
      - uses: codecov/codecov-action@v4
```
