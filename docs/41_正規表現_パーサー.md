## 1. 正規表現基本(30分)

```python
import re

# 基本
re.search(r"\d+", "abc123def")  # Match: "123"
re.findall(r"\d+", "1 abc 23 def 456")  # ['1', '23', '456']
re.sub(r"\d+", "N", "1 abc 23")  # "N abc N"

# よく使うメタ文字
\d  数字
\w  英数字+_
\s  空白
.   任意1文字
*   0以上
+   1以上
?   0または1
{n} ちょうどn個
{n,m} n〜m個
^   行頭
$   行末
[abc] a/b/cのいずれか
[^abc] それ以外
(...) グループ
(?P<name>...) 名前付きグループ
```

### 実例

```python
# メール
EMAIL = r"[\w\.-]+@[\w\.-]+\.\w+"

# 日付
DATE = r"\d{4}-\d{2}-\d{2}"

# 電話
PHONE = r"0\d{1,4}-\d{1,4}-\d{4}"

# URL
URL = r"https?://[\w\./?=&%-]+"

# IP
IP = r"\b(?:\d{1,3}\.){3}\d{1,3}\b"

# 名前付きグループ
m = re.search(r"(?P<year>\d{4})-(?P<month>\d{2})", "2026-04")
m.group("year")  # "2026"
```

## 2. ログ解析(20分)

```python
import re
from collections import Counter

LOG = re.compile(r'(\S+) - - \[(.*?)\] "(\w+) (\S+).*?" (\d+) (\d+)')

errors = Counter()
with open("access.log") as f:
    for line in f:
        m = LOG.match(line)
        if m and int(m.group(5)) >= 500:
            errors[m.group(4)] += 1

print(errors.most_common(10))
```

→ **GB級のログから5xxエラーTOP10**を1分で。

## 4. 独自形式パーサー(40分)

### Lark で文法定義

```bash
pip install lark
```

```python
from lark import Lark

grammar = """
start: expression
expression: term ("+" term)*
term: NUMBER
%import common.NUMBER
%ignore " "
"""

parser = Lark(grammar)
tree = parser.parse("1 + 2 + 3")
print(tree.pretty())
```

→ **DSL**(社内独自言語)を作れる。
