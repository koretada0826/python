## 1. 画像処理(Pillow)(20分)

### 1-1. Pillow ってなに?

> **イメージ:** Pythonで使える「**簡易フォトショ**」。
> ほとんどの画像加工は Pillow だけでOK。

```bash
pip install pillow
```

### 1-2. 基本操作

```python
from PIL import Image, ImageDraw, ImageFilter

# 画像を読み込む
img = Image.open("photo.jpg")

# サイズ変更(縦横比を保ったまま、800x800以内に収める)
img.thumbnail((800, 800))

# モード変換(RGBに統一しないと保存で失敗することがある)
img = img.convert("RGB")

# 90度回転
img = img.rotate(90)

# ぼかし(GaussianBlur=自然なぼかし、半径5px)
img = img.filter(ImageFilter.GaussianBlur(5))

# 保存(quality=85 でJPEG圧縮、ファイルサイズ抑える)
img.save("output.jpg", quality=85)
```

### 1-3. 透かし(ウォーターマーク)

```python
draw = ImageDraw.Draw(img)
# 左上(10,10)に白文字で「© 2026」を描く
draw.text((10, 10), "© 2026", fill="white")
img.save("watermarked.jpg")
```

> **使いどころ:**
> EC事業者の商品画像に**自動で透かしを入れる**バッチ処理 → 月3-5万円で受注。

## 2. OpenCV(顔検出など)(30分)

### 2-1. OpenCV ってなに?

> **イメージ:** Pillowが「**写真の編集**」なら OpenCV は「**画像の解析**」。
> 顔検出・物体検出・動画解析など**AIっぽい処理**が得意。

```bash
pip install opencv-python
```

### 2-2. 顔検出のサンプル

```python
import cv2

# 画像読み込み(OpenCVは BGR 順なので注意)
img = cv2.imread("photo.jpg")

# グレースケール化(顔検出は白黒の方が速くて正確)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 学習済みモデル(顔検出用)を読み込み
face_cascade = cv2.CascadeClassifier(
    cv2.data.haarcascades + "haarcascade_frontalface_default.xml"
)

# 顔の位置を検出(scaleFactor=1.1, minNeighbors=5)
faces = face_cascade.detectMultiScale(gray, 1.1, 5)

# 顔ごとに緑の枠を描く
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x, y), (x+w, y+h), (0, 255, 0), 2)

cv2.imwrite("detected.jpg", img)
```

### 2-3. 顔モザイク(プライバシー処理)

```python
for (x, y, w, h) in faces:
    face = img[y:y+h, x:x+w]              # 顔の部分を切り出し
    small = cv2.resize(face, (10, 10))    # 一度小さくして
    mosaic = cv2.resize(small, (w, h),    # 元サイズに戻す(=モザイク)
                        interpolation=cv2.INTER_NEAREST)
    img[y:y+h, x:x+w] = mosaic            # 元画像に貼り戻す
```

→ **YouTubeのモザイク処理代行**で月10-20万円案件あり。

## 3. OCR(Tesseract)(30分)

### 3-1. OCR ってなに?

> **イメージ:** 画像の中の**文字を読み取って文字列にする**技術。
> 「写真を見て音読する」のをAIにやらせる。

| 用途 | 例 |
|---|---|
| 請求書PDF → データ化 | 経理自動化(月10万円案件) |
| 名刺 → 顧客DB登録 | 営業ツール |
| レシート → 家計簿 | アプリ機能 |
| 古い紙資料 → 検索可能 | 法務・行政案件 |

### 3-2. インストール

```bash
brew install tesseract
brew install tesseract-lang  # 日本語対応
pip install pytesseract
```

### 3-3. 使い方

```python
import pytesseract
from PIL import Image

# 画像から日本語テキストを抽出
text = pytesseract.image_to_string(
    Image.open("invoice.jpg"),
    lang="jpn"  # 日本語指定。英語なら "eng"、両方なら "jpn+eng"
)
print(text)
```

### 3-4. よくあるエラー・落とし穴(OCR)

| 症状 | 原因 | 直し方 |
|---|---|---|
| 文字が歪む・誤認識 | 画像が傾いている | OpenCVで傾き補正してから |
| 日本語が認識されない | `lang="jpn"` 忘れ | 日本語データを `apt`/`brew` で追加 |
| 細かい文字が読めない | 解像度が低い | 画像を**2倍以上に拡大**してから |
| 背景ノイズで誤認 | カラー画像 | グレースケール+二値化で前処理 |

## 4. Whisper(音声→文字)(30分)

### 4-1. Whisper ってなに?

> **イメージ:** OpenAI が作った**最強の文字起こしAI**。
> 多言語対応、ノイズに強く、業務でほぼ実用レベル。

### 4-2. ローカル版(無料)

```bash
pip install openai-whisper
```

```python
import whisper

# モデルをロード(初回はダウンロードあり)
# tiny / base / small / medium / large(右ほど精度高いが重い)
model = whisper.load_model("base")

# 音声ファイルを文字起こし
result = model.transcribe("audio.mp3", language="ja")
print(result["text"])
```

→ **会議音声を議事録に自動変換**(月10-50万円案件)。

### 4-3. OpenAI API版(精度・速度重視)

```python
from openai import OpenAI
client = OpenAI()

with open("audio.mp3", "rb") as f:
    transcript = client.audio.transcriptions.create(
        model="whisper-1",  # OpenAI のクラウドモデル
        file=f
    )
print(transcript.text)
```

### 4-4. ローカル vs API どっち?

| 観点 | ローカル(無料) | OpenAI API |
|---|---|---|
| 料金 | 無料(電気代だけ) | $0.006/分 |
| 精度 | base〜medium ならそこそこ | 安定して高い |
| 速度 | GPU必須(CPUは激遅) | 速い |
| プライバシー | 完全ローカル | 外部送信あり |
| 案件向き | 機密情報あり/長時間 | スピード重視/短い案件 |

## 5. TTS(文字→音声)(15分)

### 5-1. TTS ってなに?

> **イメージ:** 文字を**自然な人間の声**で読み上げる技術。
> ナレーション・YouTube動画の声・電話自動応答などに使う。

```python
from openai import OpenAI
client = OpenAI()

# テキストを音声ファイルに変換
response = client.audio.speech.create(
    model="tts-1",      # tts-1(速い)or tts-1-hd(高音質)
    voice="alloy",      # alloy / echo / fable / onyx / nova / shimmer
    input="こんにちは、今日もよろしく"
)
response.stream_to_file("speech.mp3")
```

### 5-2. 用途

- **YouTube ショート**のナレーション自動化
- **電話自動応答**(「ただいま留守にしております」)
- **オーディオブック**生成
- **電子書籍 → 音声化**サービス

→ 副業で**月5-15万円**案件多数。

## 6. 動画処理(MoviePy)(15分)

```bash
pip install moviepy
```

```python
from moviepy.editor import VideoFileClip

# 動画読み込み
clip = VideoFileClip("video.mp4")

# 最初の30秒だけ切り出し
clip = clip.subclip(0, 30)

# サイズを半分に(SD化、ファイル軽量化)
clip = clip.resize(0.5)

# 書き出し
clip.write_videofile("output.mp4")
```

### 動画処理でできること

| やりたいこと | コード(雰囲気) |
|---|---|
| 切り出し | `.subclip(start, end)` |
| 結合 | `concatenate_videoclips([a, b])` |
| 字幕付け | `TextClip("...").set_position(...)` |
| 速度変更 | `.fx(vfx.speedx, 2)`(2倍速) |
| 音声差し替え | `.set_audio(new_audio)` |

→ **YouTubeショート自動生成**ツールが作れる(月10-30万円案件)。
