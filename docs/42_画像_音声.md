## 1. 画像処理(Pillow)(20分)

```bash
pip install pillow
```

```python
from PIL import Image, ImageDraw, ImageFilter

img = Image.open("photo.jpg")
img.thumbnail((800, 800))  # リサイズ
img = img.convert("RGB")
img = img.rotate(90)
img.filter(ImageFilter.GaussianBlur(5))
img.save("output.jpg", quality=85)

# 透かし
draw = ImageDraw.Draw(img)
draw.text((10, 10), "© 2026", fill="white")
```

## 2. OpenCV(顔検出など)(30分)

```bash
pip install opencv-python
```

```python
import cv2

img = cv2.imread("photo.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 顔検出
face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + "haarcascade_frontalface_default.xml")
faces = face_cascade.detectMultiScale(gray, 1.1, 5)
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x,y), (x+w, y+h), (0,255,0), 2)

cv2.imwrite("detected.jpg", img)
```

→ 顔モザイク・人数カウントなど。

## 3. OCR(Tesseract)(30分)

```bash
brew install tesseract
brew install tesseract-lang  # 日本語
pip install pytesseract
```

```python
import pytesseract
from PIL import Image

text = pytesseract.image_to_string(
    Image.open("invoice.jpg"),
    lang="jpn"
)
print(text)
```

→ **請求書PDFから自動でデータ抽出**(月10万円案件多数)。

## 4. Whisper(音声→文字)(30分)

```bash
pip install openai-whisper
```

```python
import whisper
model = whisper.load_model("base")
result = model.transcribe("audio.mp3", language="ja")
print(result["text"])
```

→ **会議音声を議事録に自動変換**(月10-50万円案件)。

### OpenAI API 経由(重い処理を任せる)

```python
from openai import OpenAI
client = OpenAI()

with open("audio.mp3", "rb") as f:
    transcript = client.audio.transcriptions.create(
        model="whisper-1",
        file=f
    )
print(transcript.text)
```

## 6. 動画処理(MoviePy)(15分)

```bash
pip install moviepy
```

```python
from moviepy.editor import VideoFileClip

clip = VideoFileClip("video.mp4")
clip = clip.subclip(0, 30)  # 最初30秒
clip = clip.resize(0.5)
clip.write_videofile("output.mp4")
```

→ **YouTube ショート自動生成**ツールが作れる。
