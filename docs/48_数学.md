## 1. 線形代数(必修)(40分)

```python
import numpy as np

# ベクトル
v = np.array([1, 2, 3])

# 行列
A = np.array([[1,2], [3,4]])

# 演算
A @ v       # 行列×ベクトル
A.T         # 転置
np.linalg.inv(A)  # 逆行列
np.linalg.eig(A)  # 固有値・固有ベクトル

# 内積
np.dot(v, v)

# ノルム
np.linalg.norm(v)
```

→ Embedding の類似度 = **ベクトル内積**。CNN = **行列演算**。

## 2. 微分・最適化(30分)

### 勾配降下法

```python
# 損失関数を最小化する
def loss(w):
    return (w - 3)**2

w = 0
lr = 0.1
for _ in range(100):
    grad = 2*(w - 3)
    w = w - lr * grad

print(w)  # 約 3.0
```

→ ML の学習 = **損失関数の勾配を下る**だけ。

### autograd(自動微分)

```python
import torch
x = torch.tensor(2.0, requires_grad=True)
y = x**2 + 3*x + 1
y.backward()
print(x.grad)  # 7.0
```

## 3. 確率(20分)

# 正規分布
stats.norm.pdf(0, loc=0, scale=1)  # 確率密度

# サンプリング

## 4. 情報理論(15分)

```python
import numpy as np

def entropy(p):
    return -np.sum(p * np.log2(p + 1e-10))

# 交差エントロピー(分類の損失)
def cross_entropy(y_true, y_pred):
    return -np.sum(y_true * np.log(y_pred + 1e-10))
```

→ ML の分類は**交差エントロピーを最小化**する作業。
