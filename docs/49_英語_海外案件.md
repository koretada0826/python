# Day 49:英語・海外案件 - 単価2-5倍

## 💡 これは何?(小学生でもわかる説明)

**Upwork等で USD建て**で働く

### なぜ大事?
同じ仕事で**2-5倍の単価**。月3000ドル=月45万円から。

日本のクラウドワークスで「Excel自動化 5,000円」がアメリカでは「**$200**」。
**同じ仕事で6倍**になる。
これは技術力が要らない。**英語の壁を越えるだけ**。

「英語できないから無理」と思ってる人が99%だから、
**ちょっと頑張った人だけが市場を独占**できる。
しかも今は**クロードが翻訳してくれる**ので、英語の壁が**過去最低**。

### イメージ
- 日本円で月100万 = USD で**月3000-5000ドル**(同じ仕事で時給2倍)
- TOEIC 600 でも**読み・書き・Slack**は十分こなせる
- 完璧な英語じゃなくて**誠意と納期**で通る

### 比喩でひとこと
日本の市場 = **狭い池**(競合多数・単価ジリ貧)
海外市場 = **広い海**(競合は世界中だが**需要も10倍**)
英語ができれば、**池から海へ船を出せる**。

### Claude Code に何を頼めばいいか
- 「Upworkプロフィール文を英語で書いて」
- 「この案件への提案文を英語で書いて(以下要件)」
- 「クライアントの英語メールを翻訳・要約して」
- 「この英語Slack 返信、丁寧かつ簡潔に」

英語ライティングは**99%クロードに任せる**。

## ⏱ 20時間

**日本円月100万 = USD月3,000-5,000**(時給4,000円)。

| 観点 | 日本 | 米国 |
|---|---|---|
| Pythonエンジニア時給 | 3,000-5,000円 | $50-150($7,500-22,500/週40h) |
| クライアント予算感 | 「安く」 | 「速く・正確に」 |
| 円安効果 | (なし) | **USD収入は資産防衛に最強** |
| 競合 | 日本人 | 世界中(でも質が低い人も多い) |

→ 同じ仕事で **2-5倍**は嘘じゃない。

## 3. プロフィール文(英語)(30分)

### 3-1. クロードに書かせる

```
Upwork のプロフィール文を英語で書いて。

私の特徴:
- Python (3+ years)
- AI/LLM development (RAG, Agent, Fine-tuning)
- Web scraping & automation
- FastAPI / Streamlit / SaaS development
- Available 30 hours/week
- Rate: $50/hour
- Time zone: JST (UTC+9)

Tone: confident, professional, results-oriented
Length: 500-800 words
Include: portfolio links, past project highlights
```

→ クロードがプロっぽい英文を生成。

### 3-2. プロフィール文の構造

```
1. Hook(冒頭1-2文で目を引く)
   "I help businesses automate workflows with AI, saving them 20+ hours weekly."

2. Skills(具体的に)
   - Python, FastAPI, PostgreSQL
   - LLM: Claude, GPT, RAG, LoRA Fine-tuning
   - Browser automation: Playwright

3. Past projects(数字で語る)
   - "Built an AI assistant for X company, reducing support tickets by 40%."

4. CTA(行動喚起)
   "Let's discuss how I can help your project. Available for a 15-min call."
```

## 4. 提案文(英語テンプレ)(20分)

```
Hi [Client name],

I read your job description carefully. You need [X], 
and I can deliver it in [Y days].

My approach:
1. [Step 1 with specifics]
2. [Step 2]
3. [Step 3]

Past projects similar to yours:
- [GitHub URL]: [1-line description]
- [Live demo URL]: [1-line description]

I'm available to start immediately. 
Happy to do a 15-min call to discuss further.

Best regards,
[Your name]
```

### コツ
- **300語以内**(忙しいクライアントは長い提案文を読まない)
- **具体的な手順**を3つ書く(やる気が伝わる)
- **過去実績**にリンク(信頼の証拠)
- **すぐ着手可能**を強調(競合に勝つ)

## 5. ビデオ会議の英語(30分)

### 5-1. 必須フレーズ

```
- "Could you repeat that, please?"           聞き取れない時
- "Let me make sure I understand. You want X, right?"  確認
- "I'll send you a follow-up email with details."  詳細はメールで
- "I'm not sure, but I'll find out and get back to you."  分からない時
- "Sorry, I missed that. Could you say it again?"  聞き逃した時
- "That sounds great. Let's go with that approach."  賛成
- "I have a concern about [X]. Could we discuss?"  懸念点
- "Let me share my screen."                  画面共有
```

→ **完璧じゃなくてOK、誠意で通る**。
むしろ「**確認のクセ**」がプロっぽく見える。

### 5-2. 練習方法

- **YouTube**:Tech系ポッドキャスト(Lex Fridman, AI Engineer など)を毎日30分
- **シャドーイング**:1日10分でOK
- **Claude Voice**:対話練習(発音矯正含む)

## 6. 海外との時差・支払い(20分)

### 6-1. 時差(ピーク時間調整)

| 地域 | 時差 | 日本側 ↔ 現地 |
|---|---|---|
| 米国西海岸(SF/LA) | -16〜-17時間 | 朝6時 = 米国前日 13-14時 |
| 米国東海岸(NY) | -13〜-14時間 | 朝8時 = 米国前日 18-19時 |
| 欧州(ロンドン) | -8〜-9時間 | 朝7時 = 欧州 22-23時 |
| 欧州(ドイツ) | -7〜-8時間 | 朝8時 = 欧州 0-1時 |

→ **米国は早朝、欧州は夕方**にミーティング集中。
完全リモートなら**自分の生活リズム**に合わせて選べる。

### 6-2. 支払い(USD受取)

| サービス | 手数料 | 特徴 |
|---|---|---|
| **Wise**(旧 TransferWise) | 安い(0.5-1%) | **おすすめ**、USD口座持てる |
| **Payoneer** | 中(1-2%) | Upwork 標準・出金まで早い |
| **PayPal** | 高め(2-4%) | 手数料高いが普及度高い |
| **Stripe Atlas** | 法人化向き | 米国法人立てて受取 |

→ **Wise で USD 受取 → そのまま JPY 出金 or USD 維持**。
円安時代は USD 維持が**資産防衛**になる。

### 6-3. 税金(必読)

- 海外収入も**日本に税金**払う必要あり(居住者の場合)
- 確定申告は**雑所得 or 事業所得**
- 経費計上できるものは漏れなく(クロード代・ネット代・PC代…)
- **クロードに税理士を紹介して**もらうのが手っ取り早い

## 8. よくある不安と対処

| 不安 | 対処 |
|---|---|
| 英語が心配 | クロード翻訳を徹底活用、文章ベースの仕事を選ぶ |
