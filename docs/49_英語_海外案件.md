## 6. 海外との時差・支払い(20分)

### 時差(ピーク時間調整)

- 米国西海岸: 17時間遅れ → 朝6時 = 米国前日昼1時
- 米国東海岸: 14時間遅れ
- 欧州: 8時間遅れ → 朝7時 = 欧州 23時

→ **米国は早朝、欧州は夕方**にミーティング集中。

### 支払い(USD受取)

- **Wise**(旧 TransferWise):手数料安い
- **Payoneer**:Upwork 標準
- **PayPal**:手数料高め

→ Wise で USD 受取 → そのまま JPY 出金 or USD 維持。
