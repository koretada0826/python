## 最速到達のコツ(全部やる)

### 1. **領域を1日目に決める**

「全部できます」より「**EC事業者向けPython自動化**」の方が刺さる。

### 2. **半額で実績作り、その後即値上げ**

最初3件は半額 → 4件目から通常 → 10件目から1.5倍

### 3. **継続案件を最優先**

単発 1万円 × 30件 = 月30万(疲弊)
月額 5万 × 6社 = 月30万(楽、安定)

→ **2件目から「月額契約しませんか?」**を毎回提案。

### 4. **同時並行3-5件**

1件ずつだと**待ち時間で死ぬ**。常時5件提案中・3件進行中。

### 5. **Twitter 1年継続**

半年でフォロワー1000人 → **毎月数件の直接依頼**。

### 6. **クロードに営業文書かせる**

提案文1件作るのに**5分**。手作業は時間の無駄。

### 7. **報告・連絡を24時間以内**

これだけで**評価★5率 90%**。

### 8. **失敗を恐れずに応募する**

採用されない = 学び。**応募 100件 = 受注 5-10件**。
