# Day 51:案件選別ガイド - 即金最速の選び方

## ⚡ 即金重視
## ⏱ 1.5時間

「**応募してはいけない案件**」を避けるだけで効率3倍。

---

## 🚦 信号別判定

### 🟢 GO(積極応募)

- 仕様**明確**(input/output/件数明示)
- 予算 **5,000円以上**(初回は1万円以上)
- クライアント評価 ★4以上(またはなし)
- クライアント本人確認済
- 締切 **1週間以上**先
- 応募数 **5-20件**

### 🟡 注意(見極め必要)

- 仕様あいまい → 質問してOKなら応募
- 予算 5,000円未満 → 量で稼げるなら応募
- 評価ない → メッセージで雰囲気確認

### 🔴 NG(絶対NG)

- 「直接連絡先教えて」(規約違反 + 詐欺)
- 「テスト納品してから判断」(踏み倒し)
- 完全成果報酬(工数掛けて0円リスク)
- 1記事500円とか(時給500円)
- 「コピペで月10万!」(スパム)
- 「月100時間で月3万」(時給300円)
- 「日本語怪しい依頼者」(海外詐欺)

---

## 単価判断フロー

```
時給換算 = 提示金額 ÷ 想定工数

時給5,000円以上 → GO
時給3,000-5,000円 → ★0時のみGO
時給1,000-3,000円 → 評価作りの最初3件のみ
時給1,000円未満 → NG
```

例:「Excel統合スクリプト 5,000円」
→ クロード使えば1時間で完成 → **時給5,000円 → GO**

例:「Web スクレイピング 月10サイト 月3,000円」
→ 月50時間掛かる → 時給60円 → **絶対NG**

## クライアントの見極め方

### 質問してみる

```
ご依頼ありがとうございます。
納品物を明確にするため、以下を教えてください:

1. データ件数の想定(10件 / 100件 / 1000件)
2. 出力形式(Excel / CSV / JSON)
3. 納期の優先度

返信の質で**良い客か悪い客**が分かる。
```

→ 返信遅い・要点ずれてる人 = 進行で揉める。**降りる勇気**。

## 危険な案件の見抜き方

### 詐欺パターン

```
タイトル:「コピペするだけで月10万」
本文:「LINE @xxx で詳細送ります」

→ 100% 詐欺。連絡先請求は規約違反。
```

```
タイトル:「Python講師募集 月20万」
本文:「教材は当社が用意」

→ 自社情報商材の販売を任される系。要注意。
```

### 踏み倒しパターン

```
「完成したら払います」
「テスト納品で見せて」

→ 仮払いない案件は受けない。ランサーズの仮払い必須。
```

## クレーム対処

### よくあるクレーム

#### 1. 「動かないんですけど」

```
動作確認の手順を教えてください:
1. Pythonバージョン (python3 --version)
2. 実行コマンド
3. エラーメッセージのスクショ

これがあれば即解決します。
```

→ **冷静に質問**で時間稼ぎつつ確認。

#### 2. 「想定と違う」

```
ご依頼当時の仕様書を確認しました。
〇〇の動作で間違いありませんか?

仕様変更でしたら追加見積もりさせていただきます。
```

→ **当初契約と違う要求**は別案件として見積。

#### 3. 「★1付けます」

```
ランサーズ運営に通報。
契約完了して支払い受けてから対応。
```

→ 規約違反は**運営に依頼**。
