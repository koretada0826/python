## 1. Resend(モダン・最推奨)(20分)

```bash
pip install resend
```

```python
import resend

resend.api_key = os.getenv("RESEND_API_KEY")

resend.Emails.send({
    "from": "noreply@example.com",
    "to": ["user@example.com"],
    "subject": "ようこそ",
    "html": "<h1>登録完了</h1><p>サービスへようこそ</p>",
})
```

→ **無料プラン月3,000通**。SendGridより簡単。

## 2. テンプレート(Jinja2)(15分)

```python
from jinja2 import Environment, FileSystemLoader

env = Environment(loader=FileSystemLoader("emails"))
template = env.get_template("welcome.html.j2")

html = template.render(name="太郎", url="https://app.com/start")
resend.Emails.send({"from": "...", "to": ["..."], "subject": "...", "html": html})
```

`emails/welcome.html.j2`:
```html
<h1>{{ name }}さん、ようこそ</h1>
<p><a href="{{ url }}">サービスを開始する</a></p>
```

## 6. LINE Messaging API(15分)

```bash
pip install line-bot-sdk
```

```python
from linebot.v3.messaging import Configuration, ApiClient, MessagingApi, PushMessageRequest, TextMessage

config = Configuration(access_token=os.getenv("LINE_TOKEN"))
with ApiClient(config) as api_client:
    api = MessagingApi(api_client)
    api.push_message(PushMessageRequest(
        to="USER_ID",
        messages=[TextMessage(text="お知らせ")]
    ))
```

→ B2C SaaS で**プッシュ率70%**(メールより高い)。
