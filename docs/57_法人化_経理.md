# Day 57:法人化・経理深掘り - 月100万後の節税

## 💡 これは何?(小学生でもわかる説明)

**会社を作って、税金を減らす**話。

### なぜ大事?
- 個人のままだと、月100万稼ぐと**税金で半分近く持っていかれる**
- 法人化(会社化)すると、**手取りが月+10万**増えるケースが普通
- 月100万安定したら**法人化はほぼ義務**(知らないと損する)
- 同じ売上でも **法人 vs 個人で年100万円以上差**が出る

### イメージ
- **個人事業主の財布** = 普通の財布
  - 稼いだお金=全部「あなたの所得」になる
  - 所得税・住民税が**最大55%**かかる(超痛い)
- **法人の財布** = 節税できる魔法の財布
  - 稼いだお金は「会社のお金」になる
  - 自分には**役員報酬(月給)**として一部だけ渡す
  - 「会社のお金」と「あなたのお金」を分けて**両方の税率を下げる**ことができる

> **もう1つの例え:** 
> 個人=ひとつの大きいバケツ(雨が全部かかる=税金重い)
> 法人=**バケツを2つに分ける**(雨を分散=税金軽い)

### Claude Code に何を頼めばいいか
- 「年商1200万の場合の個人 vs 法人の税額シミュレーションして」
- 「合同会社の設立費用と必要書類リストアップして」
- 「役員社宅制度の節税効果を試算して」
- 「マネーフォワード会社設立の手順をまとめて」

## 1. 法人化のタイミング(=「いつ会社にする?」)(15分)

### 1-1. 目安

- 年商 **800〜1,000万円超**
- 月収 **安定して80万円超**
- 来期も同じペースで稼げる**見込み**

→ 「**毎月100万円が3ヶ月続いた**」が判断ポイント。

> **なぜこのライン?**
> - 年商1,000万円超で**消費税の課税事業者**になる(個人/法人どちらも)
> - 法人化のコスト(設立費・税理士費)を**節税効果がカバー**できる金額
> - これ以下で法人化すると**コスト負け**する

### 1-2. 法人化のメリット

| 項目 | 個人 | 法人 |
|---|---|---|
| 所得税 | 最大55% | **約30%(法人税)** |
| 経費の幅 | 限定的 | **広い(社宅・出張日当)** |
| 信用 | 低い | **高い(月100万案件取りやすい)** |
| 退職金積立 | 不可 | **可(中小退職金共済)** |
| 家族への分配 | 制限 | **役員報酬で所得分散OK** |
| 赤字繰越 | 3年 | **10年** |

### 1-3. デメリットも知っておく

- 設立コスト 約25万円(自力)
- 税理士費 月3〜5万円
- 赤字でも法人住民税 約7万円/年
- 会計が複雑になる(会計ソフト必須)

> **コツ:** メリットがデメリットを**確実に上回る金額**=月100万安定。

## 3. 役員報酬の設計(=「自分への給料の決め方」)(20分)

### 3-1. 原則(超大事)

- **月額固定**(年度内変更不可)
- 期首から**3ヶ月以内に決定**
- 変えるとペナルティ(損金算入されない)

> **なぜ固定?**
> 役員報酬を自由に変えられると「税金の少ない年だけ高くする」ができてしまう。
> 国はそれを防ぐため**期首3ヶ月以内固定ルール**を設けている。

### 3-2. 最適化シミュレーション(年商1200万円の例)

```
売上              1,200万円/年
- 経費              200万円
- 役員報酬          600万円(月50万)  ← 個人税率は20%程度
─────────────────────────────────
= 法人利益          400万円
- 法人税            100万円(約25%)
─────────────────────────────────
= 内部留保(会社残り) 300万円
```

> **ポイント:** 
> 個人と法人で所得を**分散して総税率を下げる**。
> 役員報酬を高くしすぎる→個人の所得税が増える。
> 低くしすぎる→法人税が増える。
> **ちょうどいい配分**を税理士と決める。

### 3-3. 一般的な配分目安

| 年商 | 役員報酬目安(月) |
|---|---|
| 1,000〜1,500万 | 40〜50万 |
| 1,500〜2,500万 | 60〜80万 |
| 2,500〜5,000万 | 80〜130万 |

## 4. 経費にできるもの(=「個人より広がる経費の世界」)(20分)

| 法人で経費OK | 例 |
|---|---|
| **役員報酬** | 自分への給与(年600万等) |
| **役員社会保険** | 健康保険・厚生年金(半分は会社負担) |
| **退職金積立** | 中小企業退職金共済 |
| **出張日当** | 1日5,000〜10,000円(個人なら不可) |
| **社宅** | 自宅を借り上げ社宅化(家賃50%程度) |
| **旅行・接待** | 福利厚生で年5万まで |
| **生命保険** | 一部経費化(全額損金タイプ等) |
| **法人カード会費** | 全額 |
| **家族への給与** | 家族役員に役員報酬支給で所得分散 |

→ **個人で経費にできなかったもの**が法人なら可能。

> **重要:** 全部を闇雲に経費化はNG。
> **「事業に必要」な合理性**がないと税務調査で否認される。
> **税理士と相談**しながら線引きする。

## 6. インボイス対応(法人)(=「請求書のルール」)(15分)

### 6-1. 概要

法人化すると**自動的に課税事業者**(売上1,000万円超なら)。

→ **インボイス番号(T で始まる13桁)** を発行 → 請求書に明記。

### 6-2. やること

1. 法人設立後、税務署にインボイス登録申請
2. 番号取得(T1234567890123 形式)
3. 請求書テンプレに番号を記載
4. クライアントに番号を伝える

> **なぜ?**
> インボイス番号がないと、クライアント側が**仕入税額控除できない**。
> = クライアントが損するので、契約打ち切られるリスク。

## 8. 役員社宅・退職金(=「節税の二大兵器」)(20分)

### 8-1. 役員社宅(節税効果デカい)

```
家賃20万円のマンション → 個人借契約だと20万自腹

法人借契約 → 役員社宅
- 法人が大家に20万支払い(経費計上)
- 役員(あなた)は法人に5〜10万支払い(法定計算式)
- 差額10〜15万が節税効果(=会社の経費として落とせる)
```

→ **月10〜15万の手取りUP**。年間で120〜180万。

> **計算式の概要:**
> 役員から徴収する家賃 = 床面積 × 12円 + 借家床面積 × 50% など、
> 国税庁の式で算出。**家賃の10〜30%で住める**。

### 8-2. 退職金(超優遇税制)

- **中小企業退職金共済**:月7万まで全額損金、20年で1500万円
- **小規模企業共済**:月7万まで全額所得控除(個人/法人代表向け)
- 退職時の税金は**一般所得の半分以下**(退職所得控除あり)

> **これ知らないと損する制度No.1。**
> 月7万 × 12ヶ月 × 20年 = 1,680万円が**ほぼ非課税**で受け取れる。
