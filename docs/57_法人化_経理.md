# Day 57:法人化・経理深掘り - 月100万後の節税

## 💡 これは何?(小学生でもわかる説明)

****会社を作って税金を減らす****

### なぜ大事?
月100万後、法人化で**手取り月+10万**

### イメージ
個人=普通の財布 / 法人=**節税できる財布**

### Claude Code に何を頼めばいいか

## 💰 節税で +10-30万円 / 月
## ⏱ 8時間

月100万到達後、**法人化で手取りが月10万円増える**。

---

## 1. 法人化のタイミング(15分)

### 目安

- 年商 800-1,000万円超
- 月収 安定して80万円超
- 来期も同じペースで稼げる見込み

→ 「**毎月100万円が3ヶ月続いた**」が判断ポイント。

### 法人化メリット

- 所得税(個人 最大55%)→ 法人税(約30%)
- 経費の幅が広がる(社宅・出張日当)
- 信用UP(月100万案件取りやすい)
- 退職金積立(中小企業退職金共済)
- 家族に役員報酬で所得分散

---

## 2. 法人設立フロー(30分)

### 必要なもの

| 項目 | 費用・時間 |
|---|---|
| 定款認証 | 約5万円(電子定款で0円) |
| 登録免許税 | 15万円(株式会社) |
| 司法書士 | 5-10万円(自分でやれば0円) |
| 印鑑作成 | 1-2万円 |
| 銀行口座開設 | 無料 |
| 計 | 約**25万円**(自力)、約**40万円**(代行) |

### 自分でやる(マネーフォワード会社設立)

→ 無料で全部できる。30分入力で書類完成。

### 法人形態

- **合同会社(LLC)**:6万円で作れる、運営楽
- **株式会社**:25万円、信用度高い

→ **個人フリーランスなら合同会社で十分**。

---

## 3. 役員報酬の設計(20分)

### 原則

- 月額固定(年度内変更不可)
- 期首から3ヶ月以内に決定
- 高すぎ→所得税、低すぎ→法人税で圧迫

### 最適化目安(月商100万円なら)

```
売上     1,200万/年
- 経費    200万
- 役員報酬 600万(月50万)  ← 個人税率20%程度
= 法人利益 400万
- 法人税   100万(25%)
= 内部留保 300万
```

→ **個人と法人で所得を分散**して総税率を下げる。

---

## 4. 経費にできるもの(個人より広い)(20分)

| 法人で経費OK | 例 |
|---|---|
| 役員報酬 | 自分への給与 |
| 役員報酬の社会保険 | 健康保険・厚生年金(半分は会社負担) |
| 退職金積立 | 中小企業退職金共済 |
| 出張日当 | 1日5,000-10,000円(個人なら不可) |
| 社宅 | 自宅を借り上げ社宅化(家賃50%) |
| 旅行・接待 | 福利厚生で年5万まで |
| 生命保険 | 一部経費化 |
| 法人カード会費 | 全額 |

→ **個人で経費にできなかったもの**が法人なら可能。

---

## 5. 税理士の選び方(15分)

### 探し方

- 知人紹介(一番安心)
- マネーフォワード/freee の認定税理士
- 年間契約 月3-5万円が相場

### 選ぶ基準

- IT業界に詳しい
- メール/Slack 対応OK(電話のみは古い)
- クラウド会計 freee or マネフォ対応
- 領収書スマホ撮影でOK

## 6. インボイス対応(法人)(15分)

法人化すると**自動的に課税事業者**(売上1,000万円超なら)。

→ インボイス番号(T で始まる13桁)発行 → 請求書に明記。

## 8. 役員社宅・退職金(20分)

### 社宅(節税効果デカい)

```
家賃20万円のマンション → 個人借契約だと20万自腹

法人借契約 → 役員社宅
- 法人が大家に20万支払い(経費)
- 役員は法人に5-10万支払い(法定計算式)
- 差額10-15万が節税効果
```

→ 月10-15万の手取りUP。

### 退職金(超優遇税制)

- 中小企業退職金共済:月7万まで全額損金、20年で1500万
- 退職時の税金は一般所得の半分以下
