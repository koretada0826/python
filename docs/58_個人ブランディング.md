## 1. ポジショニング(20分)

「**何の人**」と思われたいか。

### 例

❌ 「**Pythonエンジニアです**」 → 100万人いる
✅ 「**EC事業者向けに業務自動化やってます、月20時間削減できます**」 → 唯一無二

### 公式: ポジショニング = 業界 × 専門技術 × 成果

```
[業界] EC / 不動産 / 医療 / 士業 / SaaS
×
[技術] AI / 自動化 / 分析 / システム
×
[成果] 削減時間 / 売上UP / コスト削減
```

→ あなたなら何が刺さる?

## 2. SNS 戦略(深掘り)(40分)

### Twitter/X 投稿の型

| 型 | 例 | 反応 |
|---|---|---|
| **数字付き成果** | 「クライアント年間60万削減した自動化」 | ★★★★★ |
| **失敗談** | 「初案件で◯◯ミスった話」 | ★★★★ |
| **具体tips** | 「pandas で複数Excel統合する3行コード」 | ★★★★ |
| **質問・問いかけ** | 「みんなどのDBクライアント使ってる?」 | ★★★ |
| **共感** | 「フリーランスあるある」 | ★★ |

### バズる構造

```
[1行目] 結論・驚き
↓
[2-5行目] 詳細・数字
↓
[最終行] CTA(リプ・リンク・続編予告)
```

例:
```
クライアントの月20時間の作業を Python で自動化したら、
1分で終わるようになった。

人件費換算で月60,000円削減。
→ 年間72万円のコストカット。

開発期間: 5日
工数: 計15時間
コード: 約400行

詳細をブログに書いた → [URL]
```

## 3. note ブログ戦略(30分)

### 月10記事ルール

- SEO狙い: 「Python 〇〇 自動化」「FastAPI 〇〇」
- ストーリー: 「初案件で〇〇した話」
- ノウハウ: 「AI Bot 構築 5ステップ」

### 記事の型

```
1. タイトル(検索ワード入り)
2. 導入: 誰のどんな悩みを解決
3. 結論: この記事で何が分かる
4. 本論: 5-7セクション
5. まとめ + CTA(問い合わせ)
```
