# Day 59:メンタル・継続術 - 365日続ける技術

## ⚡ 月100万維持の核心
## ⏱ 10時間

「**才能じゃなく継続**」を実現する具体的方法。99%の人が脱落する理由を回避。

---

## 1. 燃え尽き対策(20分)

### NG パターン

- 月150万稼げる時に**働きすぎ → 体壊す**
- 「もっと、もっと」で**家族犠牲**
- 比較病(SNS で他人と)で**自己否定**

### OK パターン

- **週1日完全オフ**(コードもSNSも見ない)
- 月収目標達成したら**働く時間を減らす**(週30h上限)
- **健康診断 / 歯科** 定期検査
- **運動週3日**(散歩でOK)

---

## 2. 習慣化の科学(20分)

### 小さく始める

❌ 「毎日5時間勉強」 → 3日でやめる
✅ 「毎日 PCを開く」 → 365日続く

→ **ハードルを馬鹿げたほど低く**設定。

### トリガー設計

```
[既存習慣] → [新習慣]

例:
- 朝コーヒー飲んだ後 → Python学習15分
- 歯磨きしながら → Twitter 1投稿
- 寝る前 → progress.md 更新
```

→ **既存行動とセット**で続く。

---

## 3. 進捗の可視化(15分)

### GitHub の草を毎日生やす

```bash
# 毎日 1コミット = 緑の草
git commit -m "..." && git push
```

→ **見える化**で続く。

### 学習トラッカー

| 日 | 時間 | 内容 | できた率 |
|---|---:|---|---:|
| 4/29 | 3h | Day 1 完了 | 100% |
| 4/30 | 2.5h | Day 2 8割 | 80% |

→ 1ヶ月で振り返ると **「自分こんなに進んだ」** で継続力UP。

---

## 4. モチベ低下の対処(20分)

### 「やる気でない」時のルール

1. **15分だけやる**(始められれば乗る)
2. **超簡単な作業**から(README整理など)
3. **環境変える**(カフェ・図書館)
4. **散歩**してから戻る
5. それでも無理なら**休む**(罪悪感ゼロで)

### 比較病の処方箋

- SNSで「年収3000万」「20代CTO」見て凹む
- → **3年前の自分と比較**(過去の自分一択)
- フォローを**整理**(妬み源を切る)

---

## 5. 集中力ハック(20分)

### ポモドーロ・テクニック

- 25分集中 → 5分休憩 × 4セット → 30分大休憩
- アプリ:**Forest, Be Focused**

### Deep Work の確保

- スマホを別部屋
- Slack/メール 通知OFF
- 朝1番で2時間 = **1日の生産性70%**

### 環境

- デスクは美しく(雑然さ = ストレス)
- 椅子を投資(腰痛で死ぬ前に)
- ノイキャンイヤホン

---

## 6. 失敗との付き合い方(15分)

### よくある失敗

- ★1評価ついた → 1件だけ、99%は★5
- 案件落ちた → **平均30件中1件しか採用されない**(普通)
- バグ出した → 修正して納品 → 信頼さらに上がる

→ **失敗=成長**と再定義。

---

## 7. 健康投資(15分)

### 必須

- **睡眠 7時間以上**(生産性 = 睡眠で決まる)
- **食事**(コンビニ依存はNG)
- **運動 週3日 30分**(散歩でOK)
- **歯科 半年に1回**
- **健康診断 年1回**

### コスパ最強投資

- 椅子 5万円 → 腰痛回避 = 数百万の医療費節約
- マットレス 5万円 → 睡眠改善
- ジム月5,000円 → 運動継続

## 8. 環境設計(15分)

### 朝のルーティン

```
6:00 起床
6:05 水・軽食
6:10 散歩 30分
6:40 シャワー
7:00 Deep Work 開始(Day 学習 or 案件)
9:00 メール/Slack確認
```

→ **朝に学習終わらせる**と1日が楽になる。

### 夜のルーティン

```
21:00 仕事終了
21:30 progress.md 記入
22:00 風呂
22:30 読書 / 動画
23:00 就寝
```
